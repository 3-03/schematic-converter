# Image to Litematica Converter

Простой и быстрый инструмент для конвертации картинок в схемы построек для Minecraft (формат `.litematica`).

**[Открыть конвертер в браузере](https://3-03.github.io/schematic-converter/)**

## Что умеет:
- Конвертировать любые картинки (PNG, JPG, WEBP) в схемы размером 32х32, 64х64, 128х128 или 256х256 блоков.
- Показывать интерактивное превью: при наведении на пиксель отображается название блока Minecraft и его координаты.
- Генерировать список материалов: выводит точное количество каждого блока, отсортированное по частоте использования.
- Скачивать результат как готовый файл `.litematica` для мода или экспортировать пиксель-арт в высоком разрешении PNG.
- Работать полностью на клиенте — данные не отправляются на сервера, обработка происходит прямо в вашем браузере.

## Как устроен проект:
Вся логика написана на чистом JavaScript (Vanilla JS), без сторонних фреймворков и библиотек сборки (нет React/Webpack/Vite). Интерфейс стилизован с использованием кастомного CSS (эффект Glassmorphism с анимированным фоном).

## Локальный запуск:
Если хотите запустить проект локально:
1. Клонируйте репозиторий:
   ```bash
   git clone https://github.com/3-03/schematic-converter.git
   ```
2. Откройте `index.html` в браузере (или запустите локальный сервер, например Live Server в VS Code).
