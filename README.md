# Image to Litematica Converter 🎨➔🧱

Красивый и функциональный веб-инструмент для конвертации любых изображений в схемы Minecraft (`.litematica`). 

[Попробовать онлайн (GitHub Pages)](https://3-03.github.io/schematic-converter/)

## ✨ Особенности

- 🖼️ **Конвертация в один клик:** Просто перетащите изображение (PNG, JPG, WebP).
- 📏 **Различные разрешения:** Поддержка размеров 32×32, 64×64, 128×128 и 256×256 блоков.
- 🔍 **Интерактивное превью:** 
  - Наведите курсор на блок в предпросмотре, чтобы увидеть его название и точные координаты в схеме.
- 📋 **Список материалов:** Автоматический расчет нужного количества блоков, отсортированный по популярности, со стильными цветными превью.
- 💾 **Экспорт:** Быстрый экспорт готового файла схемы (`.litematica`) или изображения схемы в PNG.
- 💎 **Современный UI:** Премиальный дизайн в стиле Liquid Glass (жидкое стекло) с плавной анимацией, органическим фоном и темной темой.

## 🛠️ Технологический стек

- **HTML5 & Vanilla CSS** (анимации, кастомный скроллбар, размытие стекла)
- **JavaScript (ES6+)**
- Без использования тяжелых фреймворков и библиотек — быстрая загрузка и работа прямо в браузере.

## 🚀 Как запустить локально

1. Склонируйте репозиторий:
   ```bash
   git clone https://github.com/3-03/schematic-converter.git
   ```
2. Откройте файл `index.html` в любом современном браузере.
3. Либо запустите простой сервер (например, с помощью расширения Live Server в VS Code).
